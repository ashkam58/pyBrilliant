## What are Methods? 🤔

Methods (also called functions) are like magic spells - you write them once, name them, and then you can use that magic anywhere in your program! They help you organize code and avoid repetition.

## Basic Method Structure 🏗️

Every method has the same basic structure:

<JavaCodeRunner
  code={`public class MethodBasics {
    // Method to greet someone
    public static void greetPerson(String name) {
        System.out.println("Hello, " + name + "! 👋");

    }
    
    // Method to calculate area of a rectangle

        int area = length * width;

## Methods with Return Values 🔄

Methods can give you back a result:

<JavaCodeRunner
  code={`public class ReturnMethods {
    // Method that returns a calculation
    public static double calculateTip(double billAmount, double tipPercent) {
        return billAmount * (tipPercent / 100);
    }
    
    // Method that returns a string
    public static String getGrade(int score) {
        if (score >= 90) return "A+ Excellent! 🌟";
        else if (score >= 80) return "B Good job! 👍";
        else if (score >= 70) return "C Keep trying! 💪";
        else if (score >= 60) return "D Almost there! 📚";
        else return "F Study more! 📖";
    }
    
    // Method that returns true/false
    public static boolean isPrime(int number) {
        if (number <= 1) return false;
        
        for (int i = 2; i <= Math.sqrt(number); i++) {
            if (number % i == 0) {
                return false;
            }
        }
        return true;
    }
    
    // Method that finds the maximum of three numbers
    public static int findMax(int a, int b, int c) {
        int max = a;
        if (b > max) max = b;
        if (c > max) max = c;
        return max;
    }
    
    public static void main(String[] args) {
        // Using methods with return values
        double bill = 50.0;
        double tip = calculateTip(bill, 18);
        System.out.println("💰 Bill: $" + bill);
        System.out.println("💝 Tip (18%): $" + String.format("%.2f", tip));
        System.out.println("💳 Total: $" + String.format("%.2f", bill + tip));
        
        System.out.println("\\n📊 Grade Report:");
        System.out.println("Score 95: " + getGrade(95));
        System.out.println("Score 73: " + getGrade(73));
        System.out.println("Score 45: " + getGrade(45));
        
        System.out.println("\\n🔢 Prime Number Check:");
        int[] numbers = {2, 4, 7, 9, 11, 15, 17};
        for (int num : numbers) {
            if (isPrime(num)) {
                System.out.println(num + " is prime! ✨");
            } else {
                System.out.println(num + " is not prime");
            }
        }
        
        System.out.println("\\n🏆 Maximum finder:");
        int max = findMax(23, 67, 45);
        System.out.println("Max of 23, 67, 45 is: " + max);
    }
}`}

## Method Parameters 📥

Methods can accept different types and numbers of parameters:

<JavaCodeRunner
  code={`public class MethodParameters {
    // No parameters
    public static void printSeparator() {
        System.out.println("================================");
    }
    
    // One parameter
    public static void repeatMessage(String message, int times) {
        for (int i = 0; i < times; i++) {
            System.out.println((i + 1) + ". " + message);
        }
    }
    
    // Multiple parameters of different types
    public static void createProfile(String name, int age, double height, boolean isStudent) {
        System.out.println("👤 User Profile:");
        System.out.println("Name: " + name);
        System.out.println("Age: " + age + " years old");
        System.out.println("Height: " + height + " ft");
        System.out.println("Student: " + (isStudent ? "Yes 🎓" : "No 💼"));
    }
    
    // Array parameter
    public static double calculateAverage(int[] numbers) {
        if (numbers.length == 0) return 0;
        
        int sum = 0;
        for (int num : numbers) {
            sum += num;
        }
        return (double) sum / numbers.length;
    }
    
    // Variable arguments (varargs)
    public static int sum(int... numbers) {
        int total = 0;
        for (int num : numbers) {
            total += num;
        }
        return total;
    }
    
    public static void main(String[] args) {
        printSeparator();
        System.out.println("🎯 Method Parameter Examples");
        printSeparator();
        
        repeatMessage("Java is awesome! ☕", 3);
        
        System.out.println();
        createProfile("Alex", 22, 5.8, true);
        
        System.out.println();
        int[] scores = {85, 92, 78, 96, 88};
        double avg = calculateAverage(scores);
        System.out.println("📊 Average score: " + String.format("%.1f", avg));
        
        System.out.println();
        System.out.println("🧮 Sum examples:");
        System.out.println("sum(1, 2, 3) = " + sum(1, 2, 3));
        System.out.println("sum(10, 20, 30, 40) = " + sum(10, 20, 30, 40));
        System.out.println("sum(5) = " + sum(5));
    }
}`}
/>

## Method Overloading 🔄

You can have multiple methods with the same name but different parameters:

<JavaCodeRunner
  code={`public class MethodOverloading {
    // Different versions of the same method
    public static void printInfo(String name) {
        System.out.println("Name: " + name);
    }
    
    public static void printInfo(String name, int age) {
        System.out.println("Name: " + name + ", Age: " + age);
    }
    
    public static void printInfo(String name, int age, String city) {
        System.out.println("Name: " + name + ", Age: " + age + ", City: " + city);
    }
    
    // Different calculate methods
    public static int calculate(int a, int b) {
        return a + b;
    }
    
    public static double calculate(double a, double b) {
        return a + b;
    }
    
    public static int calculate(int a, int b, int c) {
        return a + b + c;
    }
    
    // Different area calculations
    public static double area(double radius) {
        return Math.PI * radius * radius;  // Circle
    }
    
    public static int area(int length, int width) {
        return length * width;  // Rectangle
    }
    
    public static double area(double base, double height) {
        return 0.5 * base * height;  // Triangle
    }
    
    public static void main(String[] args) {
        System.out.println("👥 Different printInfo versions:");
        printInfo("Alice");
        printInfo("Bob", 25);
        printInfo("Charlie", 30, "New York");
        
        System.out.println("\\n🧮 Different calculate versions:");
        System.out.println("5 + 3 = " + calculate(5, 3));
        System.out.println("5.5 + 3.2 = " + calculate(5.5, 3.2));
        System.out.println("1 + 2 + 3 = " + calculate(1, 2, 3));
        
        System.out.println("\\n📐 Different area calculations:");
        System.out.println("Circle (radius 5): " + String.format("%.2f", area(5.0)));
        System.out.println("Rectangle (4x6): " + area(4, 6));
        System.out.println("Triangle (base 8, height 5): " + String.format("%.2f", area(8.0, 5.0)));
    }
}`}
/>

## Scope and Local Variables 🎯

Variables inside methods have local scope:

<JavaCodeRunner
  code={`public class ScopeDemo {
    // This is a class variable (global scope)
    static int globalCount = 0;
    
    public static void methodOne() {
        // Local variable only exists in this method
        int localVar = 10;
        globalCount++;
        System.out.println("Method 1 - Local: " + localVar);
        System.out.println("Method 1 - Global: " + globalCount);
    }
    
    public static void methodTwo() {
        // Different local variable with same name is OK
        int localVar = 20;
        globalCount++;
        System.out.println("Method 2 - Local: " + localVar);
        System.out.println("Method 2 - Global: " + globalCount);
        
        // This would cause an error - localVar from methodOne not accessible here
        // System.out.println(localVar from methodOne); // ❌ Error!
    }
    
    public static void demonstrateScope(int parameter) {
        // Parameters are also local to the method
        int localCalculation = parameter * 2;
        
        // Both parameter and localCalculation only exist in this method
        System.out.println("Parameter: " + parameter);
        System.out.println("Local calculation: " + localCalculation);
    }
    
    public static void main(String[] args) {
        System.out.println("🎯 Variable Scope Demo");
        
        methodOne();
        methodTwo();
        demonstrateScope(15);
        
        // Local variables from methods are not accessible here
        // System.out.println(localVar); // ❌ Error!
    }
}`}
/>

## Real-World Example: Temperature Converter 🌡️

Let's create a practical temperature conversion utility:

<JavaCodeRunner
  code={`public class TemperatureConverter {
    
    // Convert Celsius to Fahrenheit
    public static double celsiusToFahrenheit(double celsius) {
        return (celsius * 9.0 / 5.0) + 32;
    }
    
    // Convert Fahrenheit to Celsius
    public static double fahrenheitToCelsius(double fahrenheit) {
        return (fahrenheit - 32) * 5.0 / 9.0;
    }
    
    // Convert Celsius to Kelvin
    public static double celsiusToKelvin(double celsius) {
        return celsius + 273.15;
    }
    
    // Get temperature description
    public static String getTemperatureDescription(double celsius) {
        if (celsius < 0) return "🧊 Freezing";
        else if (celsius < 10) return "❄️ Very Cold";
        else if (celsius < 20) return "🧥 Cool";
        else if (celsius < 25) return "🌤️ Comfortable";
        else if (celsius < 30) return "☀️ Warm";
        else return "🔥 Hot";
    }
    
    // Display all conversions for a temperature
    public static void displayConversions(double celsius) {
        double fahrenheit = celsiusToFahrenheit(celsius);
        double kelvin = celsiusToKelvin(celsius);
        String description = getTemperatureDescription(celsius);
        
        System.out.println("🌡️ Temperature Conversions:");
        System.out.println("Celsius: " + String.format("%.1f", celsius) + "°C");
        System.out.println("Fahrenheit: " + String.format("%.1f", fahrenheit) + "°F");
        System.out.println("Kelvin: " + String.format("%.1f", kelvin) + "K");
        System.out.println("Description: " + description);
        System.out.println();
    }
    
    public static void main(String[] args) {
        System.out.println("🌡️ Temperature Converter Demo\\n");
        
        // Test different temperatures
        double[] temperatures = {-10, 0, 20, 25, 35};
        
        for (double temp : temperatures) {
            displayConversions(temp);
        }
        
        // Individual conversions
        System.out.println("🔄 Individual Conversions:");
        System.out.println("100°C = " + String.format("%.1f", celsiusToFahrenheit(100)) + "°F");
        System.out.println("32°F = " + String.format("%.1f", fahrenheitToCelsius(32)) + "°C");
        System.out.println("0°C = " + String.format("%.1f", celsiusToKelvin(0)) + "K");
    }
}`}
/>

## Method Best Practices 💡

1. **Use descriptive names** - `calculateArea()` is better than `calc()`
2. **Keep methods focused** - Each method should do one thing well
3. **Use parameters instead of global variables** when possible
4. **Return values when appropriate** - Don't just print everything
5. **Keep methods reasonably short** - If it's too long, split it up
6. **Document complex methods** with comments

## Common Method Patterns 🎨

<JavaCodeRunner
  code={`public class CommonPatterns {
    
    // Validation pattern
    public static boolean isValidEmail(String email) {
        return email != null && email.contains("@") && email.contains(".");
    }
    
    // Conversion pattern
    public static String formatCurrency(double amount) {
        return "$" + String.format("%.2f", amount);
    }
    
    // Calculation pattern
    public static int factorial(int n) {
        if (n <= 1) return 1;
        int result = 1;
        for (int i = 2; i <= n; i++) {
            result *= i;
        }
        return result;
    }
    
    // Utility pattern
    public static void printHeader(String title) {
        System.out.println("=".repeat(40));
        System.out.println("  " + title.toUpperCase());
        System.out.println("=".repeat(40));
    }
    
    public static void main(String[] args) {
        printHeader("Method Patterns Demo");
        
        // Test validation
        System.out.println("📧 Email validation:");
        System.out.println("test@example.com: " + isValidEmail("test@example.com"));
        System.out.println("invalid-email: " + isValidEmail("invalid-email"));
        
        // Test formatting
        System.out.println("\\n💰 Currency formatting:");
        System.out.println("Price: " + formatCurrency(29.99));
        System.out.println("Tax: " + formatCurrency(2.40));
        
        // Test calculation
        System.out.println("\\n🔢 Factorial calculation:");
        System.out.println("5! = " + factorial(5));
        System.out.println("3! = " + factorial(3));
    }
}`}
