## What are Variables? 🎯

In Java, variables are containers that store data values. Think of them as labeled boxes where you can put different types of information.

### Basic Data Types

Java has several primitive data types:

- **int** - Whole numbers (e.g., 42, -17, 0)
- **double** - Decimal numbers (e.g., 3.14, -2.5)
- **boolean** - True or false values
- **char** - Single characters (e.g., 'A', '5', '@')
- **String** - Text (e.g., "Hello World")

## Declaring Variables 🏗️

Here's how you declare variables in Java:

<JavaCodeRunner
  code={`public class VariablesDemo {
    public static void main(String[] args) {
        // Integer variable
        int age = 25;
        
        // Double variable
        double price = 19.99;
        
        // Boolean variable
        boolean isStudent = true;
        
        // String variable
        String name = "Alice";
        
        // Print all variables
        System.out.println("Name: " + name);
        System.out.println("Age: " + age);
        System.out.println("Price: $" + price);
        System.out.println("Is Student: " + isStudent);
    }
}`}
/>

## Variable Rules 📋

When naming variables in Java:

1. **Start with a letter** (a-z, A-Z), underscore (_), or dollar sign ($)
2. **Can contain letters, digits, underscores, and dollar signs**
3. **Cannot be Java keywords** (like `int`, `class`, `public`)
4. **Case-sensitive** (`myVar` and `MyVar` are different)
5. **Use camelCase** convention (e.g., `firstName`, `totalAmount`)

## Getting User Input 🎤

Java uses the `Scanner` class to get input from users:

<JavaCodeRunner
  code={`import java.util.Scanner;

## Type Conversion 🔄

Sometimes you need to convert between data types:

<JavaCodeRunner
  code={`public class TypeConversion {
    public static void main(String[] args) {
        // Automatic conversion (widening)
        int num = 100;
        double decimal = num;  // int to double
        
        // Manual conversion (narrowing)
        double pi = 3.14159;
        int rounded = (int) pi;  // double to int
        
        // String conversion
        String ageStr = "25";
        int age = Integer.parseInt(ageStr);
        
        System.out.println("Original int: " + num);
        System.out.println("As double: " + decimal);
        System.out.println("Pi: " + pi);
        System.out.println("Rounded: " + rounded);
        System.out.println("Age from string: " + age);
    }
}`}
/>
