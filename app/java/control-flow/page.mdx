## Making Decisions in Code 🤔

Just like in real life, programs need to make decisions. Java provides several ways to control the flow of your program based on different conditions.

## The if Statement 🚦

The most basic decision-making structure:

<JavaCodeRunner
  code={`public class BasicIf {
    public static void main(String[] args) {
        int temperature = 75;
        
        if (temperature > 80) {
            System.out.println("It's hot outside! 🌞");
        }
        
        if (temperature <= 80) {
            System.out.println("It's nice weather today! 😊");
        }
        
        // Check age for voting
        int age = 18;
        if (age >= 18) {
            System.out.println("You can vote! 🗳️");
        }
    }
}`}
/>

## if-else Statements 🔀

When you want to do one thing OR another:

<JavaCodeRunner

## Logical Operators 🧠

Combine multiple conditions with logical operators:

- **&&** (AND) - Both conditions must be true
- **||** (OR) - At least one condition must be true  
- **!** (NOT) - Reverses the condition

<JavaCodeRunner
  code={`public class LogicalOperators {
    public static void main(String[] args) {
        int age = 20;
        boolean hasLicense = true;
        boolean hasInsurance = true;
        
        // AND operator - both must be true
        if (age >= 18 && hasLicense) {
            System.out.println("You can drive! 🚗");
        }
        
        // Multiple AND conditions
        if (age >= 18 && hasLicense && hasInsurance) {
            System.out.println("You're ready to hit the road! 🛣️");
        }
        
        // OR operator - at least one must be true
        boolean isWeekend = true;
        boolean isHoliday = false;
        
        if (isWeekend || isHoliday) {
            System.out.println("No work today! 🎉");
        }
        
        // NOT operator
        boolean isWorking = false;
        if (!isWorking) {
            System.out.println("Time to relax! 😌");
        }
    }
}`}
/>

## Comparison Operators ⚖️

Use these to compare values:

- **==** Equal to
- **!=** Not equal to
- **>** Greater than
- **<** Less than
- **>=** Greater than or equal to
- **<=** Less than or equal to

<JavaCodeRunner
  code={`public class Comparisons {
    public static void main(String[] args) {
        int playerScore = 1250;
        int highScore = 1000;
        
        if (playerScore > highScore) {
            System.out.println("New high score! 🏆");
        }
        
        String password = "secret123";
        String userInput = "secret123";
        
        // For strings, use .equals() method
        if (password.equals(userInput)) {
            System.out.println("Access granted! ✅");
        } else {
            System.out.println("Wrong password! ❌");
        }
        
        // Number comparisons
        int lives = 3;
        if (lives == 0) {
            System.out.println("Game Over! 💀");
        } else if (lives == 1) {
            System.out.println("Last chance! ⚠️");
        } else {
            System.out.println("Lives remaining: " + lives + " ❤️");
        }
    }
}`}
/>

## The switch Statement 🎛️

When you have many possible values to check:

<JavaCodeRunner
  code={`public class SwitchDemo {
    public static void main(String[] args) {
        int dayOfWeek = 3;
        
        switch (dayOfWeek) {
            case 1:
                System.out.println("Monday - Start of the work week! 💼");
                break;
            case 2:
                System.out.println("Tuesday - Getting into the groove! 📈");
                break;
            case 3:
                System.out.println("Wednesday - Hump day! 🐪");
                break;
            case 4:
                System.out.println("Thursday - Almost there! 🎯");
                break;
            case 5:
                System.out.println("Friday - TGIF! 🎉");
                break;
            case 6:
            case 7:
                System.out.println("Weekend time! 🏖️");
                break;
            default:
                System.out.println("Invalid day!");
                break;
        }
        
        // Switch with strings (Java 7+)
        String grade = "B";
        switch (grade) {
            case "A":
                System.out.println("Outstanding! 🌟");
                break;
            case "B":
                System.out.println("Well done! 👏");
                break;
            case "C":
                System.out.println("Good effort! 👍");
                break;
            default:
                System.out.println("Keep trying! 💪");
        }
    }
}`}
/>

## Nested if Statements 🪆

<JavaCodeRunner

## Real-World Example: Simple Game 🎮

Let's create a simple number guessing game:

<JavaCodeRunner
  code={`import java.util.Scanner;

public class NumberGuessingGame {
    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);
        int secretNumber = 42;
        
        System.out.println("🎯 Number Guessing Game!");
        System.out.println("I'm thinking of a number between 1 and 100...");
        System.out.print("What's your guess? ");
        
        int guess = scanner.nextInt();
        
        if (guess == secretNumber) {
            System.out.println("🎉 Congratulations! You got it!");
        } else if (guess < secretNumber) {
            System.out.println("📈 Too low! The number is higher.");
        } else {
            System.out.println("📉 Too high! The number is lower.");
        }
        
        // Provide encouragement based on how close they were
        int difference = Math.abs(guess - secretNumber);
        if (difference <= 5) {
            System.out.println("🔥 You were very close!");
        } else if (difference <= 15) {
            System.out.println("👍 Not bad, getting warmer!");
        } else {
            System.out.println("🧊 Pretty far off, but keep trying!");
        }
        
        scanner.close();
    }
}`}
/>
