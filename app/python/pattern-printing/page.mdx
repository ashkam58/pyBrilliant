### 🔺 Star Triangle (The Classic!)

Now let's make a beautiful triangle pattern!

<StarTrianglePattern />

<CodeRunner starter={`# 🔺 Building a Star Triangle!
print("🏔️ Triangle Mountain:")
print()

### 🎪 Pascal's Triangle (Advanced Magic!)

This is where math meets art!

<PascalTrianglePattern />

<CodeRunner starter={`# 🎪 Pascal's Triangle - Math Magic!
def generate_pascals_triangle(rows):
    triangle = []
    
    for i in range(rows):
        row = [1]  # Every row starts with 1
        
        if triangle:  # If we have previous rows
            # Each number is sum of two numbers above it
            for j in range(len(triangle[-1]) - 1):
                row.append(triangle[-1][j] + triangle[-1][j + 1])
            row.append(1)  # Every row ends with 1
            
        triangle.append(row)
        
    return triangle

# Generate and display Pascal's Triangle
print("🎭 Pascal's Magic Triangle:")
print()

rows = 6
triangle = generate_pascals_triangle(rows)

for i, row in enumerate(triangle):
    # Add spacing for nice pyramid shape
    spaces = " " * (rows - i - 1) * 2
    numbers = "   ".join(str(num) for num in row)
    print(f"{spaces}{numbers}")

print()
print("🔮 Each number is the sum of the two above it!")`} />

## 🔺 Level 3: Shape Patterns (Artist Mode!)

### 💎 Diamond Pattern

Let's create a sparkling diamond!

<DiamondPattern />

<CodeRunner starter={`# 💎 Diamond Pattern - Precious Gem!
print("💎 Sparkling Diamond:")
print()

size = 5

# Top half of diamond (including middle)
for i in range(size):
    spaces = " " * (size - i - 1)
    stars = "*" * (2 * i + 1)
    print(f"{spaces}{stars}")

# Bottom half of diamond
for i in range(size - 2, -1, -1):
    spaces = " " * (size - i - 1)
    stars = "*" * (2 * i + 1)
    print(f"{spaces}{stars}")

print()
print("✨ Your diamond is ready to shine!")`} />
