## 7. References (Pointers) â€” Quick Peek
In Python, variables hold **references** to objects. Two names can point to the **same list**.

```python
a = [1, 2]
b = a         # b references the same list as a
b.append(3)
print(a)  # [1, 2, 3]
print(b)  # [1, 2, 3]

b = [9]       # now b points to a NEW list
print(a)  # [1, 2, 3]
print(b)  # [9]
```

> ğŸ§  **Rule of thumb**: mutating a shared object (like a list) changes it for **all** references.

Make a **copy** if you want independence:

```python
original = [1, 2, 3]
clone = original[:]      # or list(original) or original.copy()
clone.append(999)
print(original)  # [1, 2, 3]
print(clone)     # [1, 2, 3, 999]
```
